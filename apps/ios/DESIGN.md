# Trustroots iOS design guardrail

Native iOS views must retain the existing Trustroots visual language. Use
primary Trustroots green (`#12B591`) for Trustroots actions and active states;
the established search-map colours are host yes (`#58BA58`), host maybe
(`#F2AE43`), and meetup (`#11B4DA`).

Use system backgrounds for readable native surfaces, with subtle Trustroots
green borders or fills where a branded treatment helps. Do not substitute
SwiftUI's generic `.teal` or arbitrary default accent colours for branded
Trustroots elements.
